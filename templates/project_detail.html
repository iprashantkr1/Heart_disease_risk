Heart Disease Risk Level Predictor is a website which detect risk of heart disease build using different algorithms of Machine Learning in backend. The input to our algorithm is 8 features with number values and binary values.
We use algorithms such as Linear Regression and multivariable polynomial regression to output the risk percentage which indicates the chances of having heart disease and it gives us the best accuracy of 76.8%.
We have created a website by using Html, CSS and bootstrap for taking the input of patient details and used the flask module for deploying the machine learning model and processing that data.
